    The specific model identifier (e.g., 'text-embedding-3-small', 'BAAI/bge-small-en-v1.5'). Used for cache invalidation: changing the model always triggers a full re-embed, regardless of provider name. Same model via different providers (OpenAI vs Azure vs GitHub Models) produces identical vectors — no invalidation in that case.
