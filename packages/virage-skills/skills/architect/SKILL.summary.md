# Skill: Architect (summary)

Use when: writing or updating an ADR, designing a new interface or cross-package contract, planning a refactor that changes module boundaries, evaluating architectural trade-offs.

**Full skill cost:** ~1,000 tokens. Load with `read_skill('architect')` for the complete ADR workflow.

## Key outputs
- ADR appended to `docs/ADR.md` (format: Status, Context, Decision, Consequences)
- Or: architectural analysis with explicit trade-offs and decision rationale
